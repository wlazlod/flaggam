# FlagGAM

FlagGAM: rule-basis generalized additive models for explainable tabular prediction.

From-scratch implementation of the FlagGAM method (Zhao & Welsch, arXiv:2605.31189) and the Univariate Flagging Algorithm (Sheth et al., PLOS ONE 2019, CC BY).

## Installation

```bash
pip install flaggam
```

## Development

```bash
uv sync --extra dev
uv run pytest
```
